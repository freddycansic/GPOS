# 3D Graphics Engine

### Current features:
- Batch rendering 
- 2D + 3D meshes with
  - Colour
  - Bindless textures
- Semi-interactive GUI
- Not running on specific hardware

---

### Future features:
- Keyboard input processing
- FPS camera
- Basic lighting
- Ico spheres

---

### Future future features:
- GJK collision detection
- Some form of collision response
